# Оно тебе надо — аукцион вещей, в которые никто не верил

## Описание проекта

Веб-сайт для аукциона необычных вещей, которые изначально не воспринимались серьезно, но впоследствии стали культовыми. Проект представляет собой лендинг с информацией об аукционе, галереей лотов и контактными данными.

**Отличие от предыдущих проектов:** 
- Использование современных подходов к верстке (CSS Grid и Flexbox)
- Сложная типографика с кастомными шрифтами
- Адаптивная структура с оверлеями и позиционированием элементов
- Семантическая разметка HTML5

**Технологии:**
- HTML5 с семантической разметкой
- CSS3 с Grid и Flexbox
- Кастомные шрифты Raleway (Regular и Bold)
- Оптимизированная структура файлов

## Системные требования и инструкция по запуску

### Требования:
- Веб-браузер с поддержкой HTML5 и CSS3
- Локальный сервер для корректного отображения шрифтов (рекомендуется)

### Структура файлов:
```
project/
├── index.html
├── styles/
│   ├── global.css
│   └── style.css
├── fonts/
│   ├── fonts.css
│   ├── Raleway-Regular.woff2
│   ├── Raleway-Regular.woff
│   ├── Raleway-Bold.woff2
│   └── Raleway-Bold.woff
└── images/
    ├── logo-black.svg
    ├── logo-white.svg
    ├── cover.jpg
    ├── card-lot-01.jpg
    ├── card-lot-02.jpg
    ├── card-lot-03.jpg
    ├── yt.svg
    ├── vk.svg
    └── pinterest.svg
```

### Инструкция по запуску:
1. Скопируйте все файлы в одну директорию, сохраняя структуру папок
2. Запустите `index.html` в браузере
3. Для лучшей производительности рекомендуется использовать локальный сервер (например, Live Server в VS Code)
